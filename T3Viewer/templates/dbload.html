Overview
This form allows you to load the database with the telemetry data from S3.
The data is a set of temporary tables prefixed with the prefix you specify.
The current form only allows you to upload a max of a month of data. This limitation is to restrict inadvertant data load.

The allowed tags are:
  • email: e.g test2@d2.officeburrito.com. The search for devices goes back 7 days.
  • timestamp or Date/Time or Launch Time: e.g 2015-06-12T17:18:30.153Z or just use now
  • userid or user-id or User ID: eg. us-east-1:7df6e942-10c5-431f-9258-772e0b937ccd
  • deviceid or Device Id: Ncho39D6C69AXBC8EX4C81XBEB0X
  • span: e.g 2
  • search: eg. Crash or AUTOD. This can also be a regular expression like ASPC\([0-9]\)
